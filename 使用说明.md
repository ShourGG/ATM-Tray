# ATM Tray 使用说明

## 软件简介

ATM Tray 是一款轻量级后台托盘工具，用于快速激活和管理您的账号 Token。软件采用免安装设计，双击即可运行。

---

## 快速开始

### 1. 启动软件

双击 `atm-tray.exe` 启动软件，首次启动会显示登录界面。

### 2. 输入激活码

在输入框中输入您获得的激活码（格式：`XXXX-XXXX-XXXX-XXXX`），点击「激活」按钮。

### 3. 进入主界面

激活成功后进入主界面，您可以看到：
- **运行状态** - 显示当前运行状态
- **当前账号** - 显示当前激活的账号邮箱
- **剩余配额** - 显示账号剩余额度

---

## 功能说明

### 账号管理

点击底部「管理账号」按钮打开抽屉面板：

| 功能 | 说明 |
|------|------|
| **可用账号** | 显示当前激活码下的所有可用账号 |
| **激活按钮** | 点击某个账号的「激活」可切换使用该账号 |
| **刷新按钮** | 点击右上角刷新图标更新账号列表 |

### 多激活码管理

您可以添加多个激活码：

1. 打开「管理账号」抽屉
2. 在「激活码管理」区域点击 `+` 按钮
3. 输入新的激活码并确认
4. 点击激活码可切换到该激活码下的账号

### 删除激活码

在激活码列表中，点击某个激活码右侧的删除按钮，确认后即可删除该激活码。

## 界面按钮

### 标题栏按钮

| 图标 | 功能 |
|------|------|
| 🔔 小铃铛 | 查看公告信息 |
| ⚙️ 齿轮 | 打开设置 |
| ➖ 横线 | 最小化到托盘 |
| ✕ 关闭 | 关闭窗口（默认最小化到托盘） |

### 主界面按钮

| 按钮 | 功能 |
|------|------|
| 退出图标 | 退出当前激活码登录 |
| 管理账号 | 打开底部抽屉面板 |

---

## 托盘功能

关闭窗口后，软件会在系统托盘继续运行：

- **单击托盘图标** - 重新打开主窗口
- **右键托盘图标** - 显示菜单
  - 显示窗口 - 打开主窗口
  - 退出 - 完全退出程序

---

## 退出方式

点击标题栏退出图标，可选择：

| 选项 | 说明 |
|------|------|
| **正常退出** | 直接退出程序，保留登录状态 |
| **清理数据退出** | 退出并清除所有本地数据 |
| **解绑设备并退出** | 解除设备绑定，可在其他设备使用激活码 |

---

## 设置选项

点击标题栏齿轮图标打开设置：

| 设置项 | 选项 |
|------|------|
| 点击关闭按钮时 | 最小化到托盘 / 退出程序 |

---

## 自动更新

软件启动时会自动检查更新，如有新版本会弹出提示：

1. 点击「立即更新」开始下载
2. 下载完成后自动替换并重启
3. 更新过程中请勿关闭软件

## 常见问题

### Q: 激活码无效怎么办？
A: 请检查激活码是否输入正确，或联系管理员确认激活码状态。

### Q: 账号列表为空？
A: 点击刷新按钮重新获取账号列表，或联系管理员分配账号。

### Q: 如何更换设备使用？
A: 在原设备选择「解绑设备并退出」，然后在新设备输入激活码即可。

### Q: 软件无法启动？
A: 请确保系统为 Windows 10 或更高版本，并安装了 WebView2 运行时。
